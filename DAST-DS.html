Design System

Burp Suite DAST

Living reference for all design tokens, components, and patterns used in Burp Suite DAST Enterprise prototypes. When prompting for a new prototype, reference components by their section name and CSS class — e.g. .btn-primary or .age-h-over.

Colours

Two layers: Primitive palette — every shade in every colour family (the source of truth), and Semantic tokens — named by role, mapped to primitives. Always use semantic tokens in prototype code; only reference primitives when no semantic token fits.
